A development board for the new WiFi ESP32-C6 chip from SeeedStudio (Xiao line), featuring a DHT11 temperature sensor, a LDR light sensor and a WS2812b RGB LED. It can be mounted to a breadboard on a single line, taking up less space.
More info at: https://fritzenlab.com.br/fritzenlab-esp32-c6-dev-board/ or https://fritzenlab.net/fritzenlab-esp32-c6-dev-board/
